Baseline

Baseline row — appears/disappears with the anomaly modes. Now sits below the trend/LOESS sliders so it doesn't interrupt the chart–controls flow. Separated by a subtle top border.

Strict by default — the fallback is never on by default. Decade ▾ and Year ▾ open a two-option dropdown: Strict (exclude stations without the chosen period) or Nearest match (substitute the closest available period). When nearest match is active the button label changes to Decade ≈ / Year ≈. Auto modes are always strict (they pick the period with the broadest coverage, so a fallback would be redundant).

Gradient picker — spans only the range between the first and last year/decade that has any station coverage. The gradient colour encodes coverage density: dark blue-grey = no/low coverage → gold = maximum coverage. Decade chips show the count below the label. Year chips show labels on every 10th year (plus whichever is selected/hovered); all years are individually clickable.

Left / right arrows — 44 px wide × full picker height (52 px) touch targets, so they're easy to hit with a thumb. On mobile they expand to 48 px.

Status line — shows the resolved baseline and how many stations were excluded in strict mode. E.g. 401 / 512 stations · baseline: 1985 · 111 excluded (strict)
