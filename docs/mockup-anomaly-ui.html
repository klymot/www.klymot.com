Baseline row — appears/disappears with the anomaly modes. Sits between the mode row and the controls row, matching the existing row rhythm. Buttons are slightly taller than the existing chart overlay buttons for better touch targets.

~ nearest fallback toggle — pinned outside the scroll area on the right. When on, stations without the chosen period contribute using their nearest available period instead of being excluded. Automatically enabled for the Auto modes. Grayed out for Station mode (not applicable).

Gradient picker — spans only the range between the first and last year/decade that has any station coverage. The gradient colour encodes coverage density: dark blue-grey = no/low coverage → gold = maximum coverage. Decade chips show the count below the label. Year chips show labels on every 10th year (plus whichever is selected/hovered); all years are individually clickable.

Left / right arrows — 44 px wide × full picker height (52 px) touch targets, so they're easy to hit with a thumb. On mobile they expand to 48 px.

Status line — shows the resolved baseline and, for strict mode, how many stations were excluded. E.g. 401 / 512 stations · baseline: 1985 (strict — 111 excluded)
